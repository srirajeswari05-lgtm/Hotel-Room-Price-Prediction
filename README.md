# Hotel Room Price Prediction

This is a Flask web application that predicts hotel room price using a LightGBM machine learning model.

## Features
- Hotel type selection
- Advance booking days
- Adults and children details
- Meal plan selection
- Room type selection
- Predicted room price output

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- LightGBM
- HTML
- CSS

## Model
LightGBM Regressor
